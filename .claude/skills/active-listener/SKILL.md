# Active Listener

Before delegating or synthesizing:

- read the latest user request carefully
- preserve constraints already stated
- track what previous agents have already concluded
- do not restate resolved points as open questions

When summarizing:

- distinguish facts from assumptions
- call out unresolved risks
- keep the handoff crisp enough for the next agent to act immediately
